# my_laser_outputs

- Prints laser scan data as a index and range.

## Run
```bash
cd catkin_ws

catkin_make 

rosrun my_laser_outputs laser_outputs
```

### Alternative way to print

```bash
# to investigate from command line, print laster scan topic
rostopic echo /scan
```
